# 라이벌 캐릭터 이미지 파일 매핑

## 폴더 기준

- `64x64`: 실제 게임에서 사용하는 64x64 PNG 이미지
- `original_named`: 생성 원본 해상도 PNG, 요청 파일명 기준으로 정리
